# AppPilot Client Pitch Website

This is a standalone static website for pitching AppPilot to clients.

## Open Locally

Open `index.html` in a browser, or run a simple local server from this folder:

```powershell
python -m http.server 4177
```

Then visit:

```text
http://localhost:4177
```

## Pitch Angle

AppPilot is positioned as an identity application onboarding command center for IT, IAM, and security teams. The page explains the workflow from request capture to deployment and ongoing governance across PingOne, Okta, SAML, OIDC, and OAuth applications.
